#### 介绍
卡券权益交易平台

#### 技术栈
前端：
1. Vue 2.x + UniApp
2. uView UI 2.x

后端：
1. Springboot 2.4.5
2. MySQL 5.7
3. Mybatis-Plus 3.4.2
4. Redis 3.2.0
